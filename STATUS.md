## 📊 项目架构

```
后端服务 (Node.js + Express)
    ↓
数据库 (MySQL + Redis)
    ↓
区块链 (Conflux eSpace)
    ↓
智能合约 (FarmBatchNFT)
```

## 🎨 技术栈

- **后端**: Node.js, Express, Ethers.js
- **数据库**: MySQL, Redis
- **区块链**: Conflux eSpace (EVM 兼容)
- **前端**: HTML, CSS, JavaScript (原生)
- **智能合约**: Solidity 0.8.20
